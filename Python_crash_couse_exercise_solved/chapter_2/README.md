# 2.1 Simple Message

```python
# Defining the first simple message
simple_message_1 = "Imran Khan is a Famous Leader in the World"
print(simple_message_1)

# Defining the second simple message
simple_message_2 = 'Imran Khan changed the Pakistan Insha Allah'
print(simple_message_2)

# Updating the second simple message
simple_message_2 = 'Imran Khan is our hero'
print(simple_message_2)
```
